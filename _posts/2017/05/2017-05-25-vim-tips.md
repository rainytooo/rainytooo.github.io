### 将当前文件的tab转成空格

```
:set expandtab
:%retab! [数字]                 " 如果不写数字,那么按照宽度进行转换
```

其它相关命令：
```
:set tabstop=4 设定tab宽度为4个字符
:set shiftwidth=4 设定自动缩进为4个字符
:set expandtab 用space替代tab的输入
:set noexpandtab 不用space替代tab的输入
```
